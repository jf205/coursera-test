Class IdemnecantExpr

A binary expression whose operator is idemnecant.

An idemnecant operator is an operator that yields a trivial result when applied to identical operands (disregarding overflow and special floating point values).

For instance, subtraction is idemnecant (since e-e=0), and so is division (since e/e=1).
