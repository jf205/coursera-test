Predicate getPhiLowerBounds

This predicate computes the lower bounds of a phi definition. If the phi definition corresponds to a guard, then the guard is used to deduce a better lower bound. For example:

 def:      x = y % 10;
 guard:    if (x >= 2) {
 block:      f(x)
           }

In this example, the lower bound of x is 0, but we can use the guard to deduce that the lower bound is 2 inside the block.
