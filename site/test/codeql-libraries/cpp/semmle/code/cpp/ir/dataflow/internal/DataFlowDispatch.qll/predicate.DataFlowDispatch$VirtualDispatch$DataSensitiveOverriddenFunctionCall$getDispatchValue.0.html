DataSensitiveOverriddenFunctionCall

Member predicate VirtualDispatch::DataSensitiveOverriddenFunctionCall::getDispatchValue

Gets the node whose value determines the target of this call. This node could be the qualifier of a virtual dispatch or the function-pointer expression in a call to a function pointer. What they have in common is that we need to find out which data flows there, and then it’s up to the resolve predicate to stitch that information together and resolve the call.
