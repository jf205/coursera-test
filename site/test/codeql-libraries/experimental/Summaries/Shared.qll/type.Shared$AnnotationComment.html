Class AnnotationComment

A comment that annotates data flow nodes as sources/sinks.

Such a comment starts with “Semmle:”, possibly preceded by whitespace, and may contain specifications of the form “source: label,config” and “sink: label, config” where again whitespace is not significant and the “,config” part may be missing.

It applies to any data flow node that ends on the line where the comment starts, and annotates the node as being a source/sink with the given flow label(s) for the given configuration (or any configuration if omitted).
