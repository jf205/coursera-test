Class TSGlobalDeclImport

A reference to a global variable for which there is a TypeScript type annotation suggesting that it contains the namespace object of a module.

For example:

import * as net_outer from "net"
declare global {
		var net: typeof net_outer
}

var s = net.createServer(); // this reference to net is an import
