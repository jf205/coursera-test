Predicate deconstructSizeExpr

Holds if sizeExpr is an expression consisting of a subexpression lengthExpr multiplied by a constant sizeof that is the result of a sizeof() expression. Alternatively if there isn’t a suitable sizeof() expression, lengthExpr = sizeExpr and sizeof = 1. For example:

malloc(a * 2 * sizeof(char32_t));

In this case if the sizeExpr is the argument to malloc, the lengthExpr is a * 2 and sizeof is 4.
