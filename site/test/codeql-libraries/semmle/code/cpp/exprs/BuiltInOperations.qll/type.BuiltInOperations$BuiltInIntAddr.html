Class BuiltInIntAddr

A C/C++ __INTADDR__ built-in operation (used by some implementations of offsetof). The operation retains its semantics even in the presence of an overloaded operator &). This is an EDG extension.

struct S {
  int a, b;
};
int d = __INTADDR__(struct S, b); // usually 4
