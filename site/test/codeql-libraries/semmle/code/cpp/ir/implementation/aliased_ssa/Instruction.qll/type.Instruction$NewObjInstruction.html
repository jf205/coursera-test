Class NewObjInstruction

An instruction that allocates a new object on the managed heap.

This instruction is used to represent the allocation of a new object in C# using the new expression. This instruction does not invoke a constructor for the object. Instead, there will be a subsequent Call instruction to invoke the appropriate constructor directory, passing the result of the NewObj as the this argument.

The result is the address of the newly allocated object.
