Class ThisAccess

A this access, for example this on line 5 in

class C {
  int Count;

  public int GetCount() {
    return this.Count;
  }
}

Note that a this access may be implicit, for example the implicit this qualifier on line 5 in

class C {
  int Count;

  public int GetCount() {
    return Count;
  }
}
