Class SpringCamelXMLBeanRef

A reference to a Spring bean in an Apache Camel route defined in a Spring beans file.

A Camel <bean> definition - which is not the same as a Spring <bean> element - specifies a Spring bean that should be called in response to messages that match the enclosing route.
