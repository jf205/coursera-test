Predicate ejbPolyCalls

Specialized version of the polyCalls(..) predicate for the use case of finding “shortest” call chains from EJBs to forbidden methods. This is the same as polyCalls(..), with two exceptions:

  • It does not consider calls into an EJB method.
  • It does not consider calls from “forbidden callables”.
