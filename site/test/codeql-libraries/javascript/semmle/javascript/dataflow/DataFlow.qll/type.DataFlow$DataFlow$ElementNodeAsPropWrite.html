Class DataFlow::ElementNodeAsPropWrite

An array element viewed as a property write; for instance, in var arr = ["first", , "third"], "first" is a write of property 0 of arr and "third" is a write of property 2 of arr.

Note: We currently do not expose the array index as the property name, instead treating it as a write of an unknown property.
