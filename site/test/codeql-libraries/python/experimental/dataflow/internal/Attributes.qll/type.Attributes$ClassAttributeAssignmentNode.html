Class ClassAttributeAssignmentNode

Represents an attribute of a class that is assigned statically during class definition. For instance

class MyClass:
    attr = value
    ...

Instances of this class correspond to the NameNode for attr, and also gives access to value by virtue of being a DefinitionNode.
